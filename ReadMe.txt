These files accompany the "TI Robot Systems Learning Kit MAX" 
Built 8/25/2022 by Valvano and Valvano

Most of the folders contain CCS projects, with the following exceptions
1) The inc folder contains shared files. For example, code you use in one lab that you will need in the next lab will be located in the inc folder. The inc folder also contains starter code used throughout the curriculum.
2) The cc3100-sdk contains CC3100 code
3) utils contains from TI code
4) TExaSdisplay is a Windows application that implements a serial terminal, oscilloscope and logic analyzer.
