There are two possible hardware connections between MSP432 to CC2650 The \#define DEFAULT selects which connection to use. These two hardware possibilities can deployed into four options. There are two projects that can be used with these four options\+: {\bfseries{Very\+Simple\+Application\+Processor\+\_\+\+MSP432}} and {\bfseries{Application\+Processor\+\_\+\+MSP432}}.
