\doxysubsection{Detailed Description}
Take ultrasonic distance measurements. 

Provide mid-\/level functions that initialize ports, start an ultrasonic sensor measurement, and finish an ultrasonic sensor measurement using the HC-\/\+SR04 ultrasonic distance sensor.~\newline
 5V regulator output connected to HC-\/\+SR04 Vcc (+5V) ~\newline
 22k top connected to HC-\/\+SR04 Echo (digital output from sensor)~\newline
 22k bottom connected to 33k top and MSP432 P6.\+6 (J4.\+36) (digital input to MSP432)~\newline
 33k bottom connected to ground~\newline
 Ground connected to HC-\/\+SR04 ground and MSP432 ground (J3.\+22)~\newline
 MSP432 P5.\+6 (J4.\+37) (digital output from MSP432) connected to HC-\/\+SR04 trigger \begin{DoxyVersion}{Version}
