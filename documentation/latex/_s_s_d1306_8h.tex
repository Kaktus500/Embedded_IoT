\doxysubsection{Detailed Description}
Provide simple I/O functions for the SSD 1306 OLED. 

Use e\+USCI\+\_\+\+A3 to send an 8-\/bit code to the 1306 64x128 pixel OLED to display text, images, or other information. \begin{DoxyVersion}{Version}
