\doxysubsection{Detailed Description}
Provide functions for the QTR-\/8\+RC reflectance sensor array. Pololu part number 3672. 

Provide functions to take measurements using a QTRX reflectance sensor array (Pololu part number 3672). This works by outputting to the sensor, waiting, then reading the digital value of each of the eight phototransistors. The more reflective the target surface is, the faster the voltage decays. The time to wait is a function of~\newline
 1) reflectance of surface~\newline
 2) electrical resistance and capacitance of the GPIO pin~\newline
 3) distance to surface~\newline
 For these reasons, you will have to experimentally verify which time works best for your system. \begin{DoxyVersion}{Version}
