\doxysubsection{Detailed Description}
Provide PWM output functions. 

Four Timer\+A0 PWM outputs \begin{DoxyRemark}{Remarks}
PWM on P2.\+4 using Timer\+A0 TA0.\+CCR1 

PWM on P2.\+5 using Timer\+A0 TA0.\+CCR2 

PWM on P2.\+6 using Timer\+A0 TA0.\+CCR3 

PWM on P2.\+7 using Timer\+A0 TA0.\+CCR4 

MCLK = SMCLK = 3MHz DCO; ACLK = 32.\+768k\+Hz 

TACCR0 generates a square wave of freq ACLK/1024 =32Hz 
\end{DoxyRemark}
