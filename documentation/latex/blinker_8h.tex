\doxysubsection{Detailed Description}
Provide low-\/level functions that control LEDs on TI-\/\+RSLK MAX robot. 

Four LED are connected to Port 8~\newline
 1) Hardware uses positive logic low current LEDs~\newline
 2) Positioned on the four corners of the robot just for fun~\newline
 3) Software outputs to these four LEDs~\newline
