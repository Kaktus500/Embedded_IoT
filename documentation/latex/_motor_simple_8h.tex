\doxysubsection{Detailed Description}
Set motor speed and direction. 

Provide mid-\/level functions that initialize ports and set motor speeds to move the robot. Simple solution that does NOT use PWM hardware. Rather the PWM outputs are created by software delays using the Sys\+Tick timer.~\newline
