\doxysubsection{Detailed Description}
Provide functions that initialize the Sys\+Tick module. 

Time delay using Sys\+Tick busy-\/wait Provide functions that initialize the Sys\+Tick module, wait at least a designated number of clock cycles, and wait approximately a multiple of 10 milliseconds using busy wait. After a power-\/on-\/reset, the MSP432 gets its clock from the internal digitally controlled oscillator, which is set to 3 MHz by default. One distinct advantage of the MSP432 is that it has low-\/power clock options to reduce power consumption by reducing clock frequency. This matters for the function \mbox{\hyperlink{group___m_s_p432_ga812160529334c18c7be6c2cbbf450467}{Sys\+Tick\+\_\+\+Wait10ms()}}, which will wait longer than 10 ms if the clock is slower. \begin{DoxyVersion}{Version}
