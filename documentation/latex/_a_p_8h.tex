\doxysubsection{Detailed Description}
Bluetooth Low Energy Application Processor. 

Abstraction layer for implementing BLE \begin{DoxyRemark}{Remarks}
see GPIO.\+c file for hardware connections 

The CC2650 is running with Simple\+NP 2.\+2 (Powersave, MRDY, SRDY signals) 

Tested with CC2650 Booster\+Pack with these two object codes\+: cc2650bp\+\_\+simple\+\_\+np\+\_\+uart\+\_\+pm\+\_\+xsbl\+\_\+mooc\+\_\+custom.\+hex, and simple\+\_\+np\+\_\+cc2650bp\+\_\+uart\+\_\+pm\+\_\+sbl.\+hex
