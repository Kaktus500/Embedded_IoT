\doxysubsection{Detailed Description}
Provide low-\/level functions that interface bump switches the robot. 

Six switches are connected to Port 4 ~\newline
 1) Hardware uses negative logic with internal pullup~\newline
 2) Positioned on the front of the robot to detect collisions~\newline
 3) Software returns 6-\/bit positive logic (1 means collision)~\newline
