\doxysubsection{Detailed Description}
Provide receive/transmit functions for EUSCI A2. 

EUSCI\+\_\+\+A2 UART is connected to exernal device like CC2650 or HC12 \begin{DoxyRemark}{Remarks}
UCA2\+RXD (VCP receive) connected to P3.\+2 

UCA2\+TXD (VCP transmit) connected to P3.\+3 

J1.\+3 from Bluetooth (DIO3\+\_\+\+TXD) to Launch\+Pad (UART RxD)\{MSP432 P3.\+2\} 

J1.\+4 from Launch\+Pad to Bluetooth (DIO2\+\_\+\+RXD) (UART TxD)\{MSP432 P3.\+3\} 

Busy-\/wait device driver for the EUSCI A2 UART output 

Interrupting device driver for the EUSCI A2 UART input 
\end{DoxyRemark}
