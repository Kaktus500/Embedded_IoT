\doxysubsection{Detailed Description}
Provide receive/transmit functions for EUSCI A0. 

EUSCI\+\_\+\+A0 UART is connected to the PC via the USB cable To be able to use printf~\newline
 1) add UART0.\+c to project~\newline
 2) \char`\"{}..\textbackslash{}inc\textbackslash{}\+UART0.\+h\char`\"{}~\newline
 3) Must be running at 48 MHz, \mbox{\hyperlink{group___m_s_p432_ga6af2755c894e5587fbc63a8a6ad084db}{Clock\+\_\+\+Init48\+MHz()}};~\newline
 4) Call \mbox{\hyperlink{group___m_s_p432_gab6d33a6e3d01b394e07ee4c0e2b75c7e}{UART0\+\_\+\+Initprintf()}} \begin{DoxyRemark}{Remarks}
