Detailed Description

Provide receive/transmit functions for EUSCI A2.

EUSCI_A2 UART is connected to exernal device like CC2650 or HC12

Remarks
UCA2RXD (VCP receive) connected to P3.2
UCA2TXD (VCP transmit) connected to P3.3
J1.3 from Bluetooth (DIO3_TXD) to LaunchPad (UART RxD){MSP432 P3.2}
J1.4 from LaunchPad to Bluetooth (DIO2_RXD) (UART TxD){MSP432 P3.3}
Busy-wait device driver for the EUSCI A2 UART output
Interrupting device driver for the EUSCI A2 UART input
