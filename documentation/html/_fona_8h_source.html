92 JST 2-pin - this is the battery input connector. It works with any of our Lipoly batteries but since the charge rate is
93 500mA (and the cellular module can spike high current draw!) we suggest our 500mAh (https://adafru.it/drL) or
94 1200mAh (http://adafru.it/258)batteries. You can also connect a JST cable here if you have other plans. See the
95 cable photo for polarity, red is + and black is gnd. (http://adafru.it/261)
96 
97 MicroUSB connector - this is the LiPoly/LiIon battery charging port. The SIM800 has a USB interface but its ONLY
98 for reprogramming the module with an expensive and unavailable IDE. So charge only! The charge rate is
99 500mA max.
100 
101 Headset jack - this is a 'standard' TRRS 3.5mm phone headset jack with stereo earphone and mono
102 microphone. Any 'iPhone' or 'Android' compatible (but not iPhone original) should work. (https://adafru.it/fbK) We
103 tried about 10 different ones, and basically the more expensive once are more comfortable and louder but our
104 official iPhone headset mic did not work for unknown reasons. Sleeve is Mic+, first ring is ground, then the
105 second ring and tip are for stereo audio. The module does not have stereo out so we just tied both together.
106 
107 Vio - THIS IS THE MOST IMPORTANT PIN! This is the pin that you MUST drive with an external voltage from 3V-
108 5V to set the logic level converter. The converter also buffers the indicator LEDs so NOTHING will appear to
109 work unless this pin is powered! You should set the voltage to whatever voltage your microcontroller uses for
110 logic. A 5V micro (like Arduino) should have it be 5V, a 3V logic micro should set it to 3V.
111 
112 VBAT & GND - these are the same pins as the 2-pin JST connector. Connect to a 3.7V/4.2V Lipoly/LiIon battery.
113 Do not power from a power supply or LDO. Use only a battery, at least 500mA, 1000mAh+ is best since there
114 can be 2A spikes of current
115 
116 Key - This is also a super important pin (but not as important as Vio). This is the power on/off indicator. Its also
117 tied to the button in the top left. Tie this pin to ground for 2 seconds to turn the module on or off. It's not a level
118 signal so it isn't like "low is off, high is on" - instead you must pulse it for 2 seconds to turn off/on. The module
119 comes by default off. Tie this permanently to ground if you never want your micro to turn off the FONA for power
120 saving
121 
122 PS - this is the Power Status pin. It is low when the module is off and high when the module has power. If you're
123 using the Key button or pin, you can monitor this pad to see when the module's booted up. This is tied to the Pwr
124 LED too.
125 
126 NS - this is the Network Status pin. It pulses to signal the current status of the module. This is also tied to the Net
127 LED so for more detail see the LEDs section below.
128 
129 Reset - this is module hard reset pin. By default it has a high pull-up (module not in reset). If you absolutely got
130 the module in a bad space, toggle this pin low for 100ms to perform a hard reset.
131 
132 RX & TX - OK now that I made you read all that you can actually use the UART pins. The module uses UART to
133 send and receive commands and data. These pins are auto-baud so whatever baud rate you send "AT" after
134 reset or boot is the baud rate is used. RX is into the module, TX is out of the module.
135 
136 RI - this is the Ring Indicator. It is basically the 'interrupt' out pin from the module. It is by default high and will
137 pulse low for 120ms when a call is received. It can also be configured to pulse when an SMS is received.
138 
139 SPK+ and - : This is for connecting an external 8 ohm speaker, max 1W. You can configure the module to route
140 calls and FM radio to the headset or the external audio. The two pins are differential so they don't have output
141 DC blocking capacitors. You cannot connect this to a stereo, powered speakers or other non-differential
142 amplifier without adding a 100uF+ blocking cap in series to the + pin and then not using the - pin. Instead, your
143 amp should use GND for the - reference
144 
145 MIC + and -: this is for connecting an external electret microphone, it will bias the mic with 2V. Most electrets will
146 work just fine. No extra circuitry is required for the mic such as a biaser or amplifier, just wire it up directly!
147 
148 LEDs
149 PWR - Blue! Lit when the module is booted and running
150 
151 NET - Red! You can use this for checking the current state without sending an AT command:
152 - 64ms on, 800ms off - the module is running but hasn't made connection to the cellular network yet
153 - 64ms on, 3 seconds off - the module has made contact with the cellular network and can send/receive voice
154 and SMS
155 - 64ms on, 300ms off - the GPRS data connection you requested is active
156 By watching the blinks you can get a visual feedback on whats going on.
157 
158 Charging - Orange! This is next to the microUSB jack. Indicates the onboard lipo charger is charging
159 Done - Green! This is next to the JST jack. Indicates that the battery charging is done and the battery is full
160 
161 Other Breakout Pins
162 We scattered a few other breakouts around the board.
163 
164 Buzzer and PWM (Top right) - These are tied to the PWM output of the module! The PWM capability is quite nice,
165 it can set any frequency and duty cycle. The PWM pin is directly output from the module and is 0-2.8Vpp. The
166 Buzzer output has a NPN drive transistor so it can run a small vibration motor. Bz+ is the VBat voltage, Bz- is
167 toggled on and off to ground.
168 
169 ADC (left middle) - the SIM800 has an ADC that can read 0-2.8VDC from this pin, referenced to ground. It also
170 has an internal battery ADC so you can use this for a sensor or something. You can query the voltage from the
171 UART. 2.8V max, people!
172 
173 5V (left middle) - this is the USB 5V from the microUSB connector when its in and powered. Good if you need to
174 know when the microUSB is plugged in and/or want to recharge the battery from an external plug.
175 
176 2.8V test point - We have a test point for the 2.8V internal regulator, its off to the right.
177  */
